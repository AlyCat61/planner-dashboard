# Interactive Planner

An interactive web-based planner that allows users to organize tasks, schedule future activities, and manage daily productivity in a simple and intuitive interface.

---

## Overview

The **Interactive Planner** is a lightweight productivity tool designed to help users plan their day, schedule tasks ahead of time, and keep track of priorities.

This project focuses on creating a clean, interactive interface where tasks can be added, managed, and organized efficiently.

It was built as a web-based project and is intended to demonstrate practical frontend development and application design.

---

## Features

* Add tasks with a **title, date, and time**
* Schedule tasks **for future days**
* Assign **priority levels** (Low, Medium, High)
* Add **notes** to tasks
* Mark tasks as **completed**
* Delete tasks
* Filter tasks by:

  * All
  * Today
  * To Do
  * Completed
* Visual **task statistics dashboard**

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/interactive-planner.git
```

Navigate to the project folder:

```bash
cd interactive-planner
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
